# Proyecto React - Carrito

Proyecto desarrollado con React + Vite.

## Funcionalidades
- Listado de productos desde MockAPI
- Carrito con suma, resta y eliminación
- Context API
- Búsqueda de productos
- Manejo de errores y loading
- Diseño responsive
- Internacionalización (ES / EN / HE)

## Tecnologías
- React
- Vite
- React Router
- Context API
